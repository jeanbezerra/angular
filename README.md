# angular

## Setup
```sh
Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned
```
```sh
npm install -g @angular/cli
```

## Criando novos projetos

```sh
ng new project-name --minimal=true --routing=true --style=scss --ssr=true
```
```sh
ng analytics disable
```

```sh
cd project-name
```

```sh
ng serve --open
```
